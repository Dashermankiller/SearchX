Crawler Network Settings

Controls how aggressively the web crawler fetches pages. Changes take effect the next time the crawler is started.

Max concurrent requests

Total simultaneous HTTP requests across all domains. Higher = faster on fast networks.

Max per domain

Max concurrent requests to a single domain. Keep low to be polite.

Delay per domain (seconds)

Minimum wait between requests to the same domain.

DB write workers

Thread pool size for database writes. Match to your CPU core count.

Connection pool limit

Maximum total open TCP connections. Should be ≥ max concurrent.

DNS cache TTL (seconds)

How long to cache DNS lookups. Longer = fewer DNS queries.

Domain crawl budget

Maximum pages to crawl per domain per session. Prevents runaway crawls on large sites.

Capture screenshots

Save a JPEG thumbnail of each crawled page using Playwright. Requires playwright installed.

How file permissions work

  1. If the extension is in Blocked → always denied, no matter what.
  2. If the extension is not in Allowed → denied.
  3. If it passes both checks → file opens with your default app.
